| Path Enviroment   | Feature                                                                                        |
| ----------------- | ---------------------------------------------------------------------------------------------- |
| 3DMODEL_DIR       | Point to a directory that contains 3D models of components for use by the 3D viewer.           |
| 3D_PARTY          | Point to a directory that contains 3rd party plugins, libraries and other downloadable content |
| FOOTPRINT_DIR     | Points to a directory that contains footprint files for use by Pcbnew.                         |
| SYMBOL_DIR        | Points to a directory that contains symbol files for use by Eeschema                           |
| TEMPLATE_DIR      | Points to a directory that contains sheet template files for use by Eeschema.                  |
| USER_TEMPLATE_DIR | Points to a directory that contains project template files created by the user.                |
