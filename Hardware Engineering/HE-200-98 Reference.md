# How to Install new symbol
![[Pasted image 20260417103711.png]]

![[Pasted image 20260417104023.png]]
1. Click on "Download Now" to start the download.
2. Change the name of the ".lib" file to make it easier for to recognize it.
3. Prefer to change this to change this to the model of the part that symbol represent
4. Copy the folder that contains the part to a folder where keep KiCad libraries.
![[Pasted image 20260417104514.png]]
5. "Manage Symbol Libraries" from the Preference menu. 
![[Pasted image 20260417104814.png]]
![[Pasted image 20260417104933.png]]
- Click on the folder button to bring up the file browser and navigate to the location where save the new .lib file.
- The new ".lib" file now appears in a new row in the Symbol Library.
