# Inside of ESP32
![[Pasted image 20260305094833.png]]

- 3.3V voltage regulator that drops the input voltage to power the ESP32 chips.
- CP2102 Chip that allows to plug the ESP32 to computer to program without the need for an FTDI programmer.
- ENABLE button - reboots ESP32.
- Hold down the BOOT button and then press the enable, ESP32 reboots in programming mode.
