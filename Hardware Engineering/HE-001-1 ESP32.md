# What is it?
- A low-cost high performance microcontroller developed by Espressif Systems.
- It is widey used for Iot project because it has built-in WiFi and Bluetooth.
# Characteristic
- Dual-core 32-bit processor.
- Clock speed up to 240 MHz.
- Built-in WiFi and Bluetooth
- Support 
- Low power consumption
- Affordable price

# Specifications
- Dual core processor
- Wi-Fi and Bluetooth built-in
- Run 32 bit programs.
- Clock frequency can go up to 240 MHz and it has a 512 kB RAM.
- Has 30 / 36 pins.
# GPIO
## Left
- Enable (EN) - Reset / Enable button
- GPIO36 (A0) - Analog input (ADC), input only
- GPIO39 (A3) - Analog input, input
- GPIO34 / 35 - input only
- GPIO32 / 33 - Digital I /O, support ADC
- GPIO25 (DAC1) - Analog output
- GPIO26 (DAC2) - Analog output
- GPIO27 - GPIO Received all input and output
- GPIO14 - HSPI CLK
- GPIO12 - HSPI MISO
- GPIO13 - HSPI MOSI
- GND - Ground
- VIN - 5V input
## Right
- GPIO23 - VPSI MOSI
- GPIO22 - I2C SCL
- GPIO1 (TX0) - UART transmit 
- GPIO3 (RX0) -UART receive 
- GPIO2 - (I2C SDA)
- GPIO19 - VPSI MISO
- GPIO18 - VPSI CLK
- GPIO5 - VSPI CS
- GPIO17 - TX2
- GPIO16 - RX2
- GPIO4 - GPIO Received all input and output
- GPIO2 - GPIO Received all input and output
- GPIO15 - HSPI CS

## Full System Information Code
```c++
#include <WiFi.h>

void setup() {
  Serial.begin(115200);
  delay(1000);

  Serial.println("===== FULL ESP32 SYSTEM INFO =====");

  // Chip Info
  Serial.print("Chip Model: ");
  Serial.println(ESP.getChipModel());

  Serial.print("Chip Revision: ");
  Serial.println(ESP.getChipRevision());

  Serial.print("CPU Frequency (MHz): ");
  Serial.println(ESP.getCpuFreqMHz());

  // Flash Info
  Serial.print("Flash Size (bytes): ");
  Serial.println(ESP.getFlashChipSize());

  Serial.print("Flash Speed (Hz): ");
  Serial.println(ESP.getFlashChipSpeed());

  // Memory Info
  Serial.print("Free Heap: ");
  Serial.println(ESP.getFreeHeap());

  Serial.print("Min Free Heap: ");
  Serial.println(ESP.getMinFreeHeap());

  Serial.print("Max Alloc Heap: ");
  Serial.println(ESP.getMaxAllocHeap());

  // SDK
  Serial.print("SDK Version: ");
  Serial.println(ESP.getSdkVersion());

  // MAC Address
  Serial.print("WiFi MAC Address: ");
  Serial.println(WiFi.macAddress());

  Serial.println("===================================");
}

void loop() {
}
```
