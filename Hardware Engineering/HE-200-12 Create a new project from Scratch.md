Start a new project from a templates, can take advantage of work that you have done in the past.

- Create Arduino shields, can set up an Arduino shield base template and use it to create new Arduino shields project. 

KiCad will set up new project. You will see 3 new files:
- The main project files with extension ".kicad_pro"
- The schematic design file with extension ".kicad_sch"
- The layout design file with extension ".kicad_pcb"
