# What is it?
- A digital sensor that operates within a supply voltage range of 3.3V to 5V. 
- Can measure temperatures from -55°C to 125°C.
- Ideal for projects needing accurate and reliable temperature monitoring.

# Characteristic
2 type of power supply
# Normal Power Mode
   - Most stable.
   - Faster temperature reading
   - Works well with long cables
   - Supports multiple D18B20 sensors
## Wiring

| D18B20 Pin | Connection                     |
| ---------- | ------------------------------ |
| VDD        | 3.3V                           |
| GND        | GND                            |
| DQ         | GPIO + 4.7 kΩ pull-up resistor |

# Parasite Power Mode
- Less stable
- Temperature reading may fail
- Not suitable for long wires
