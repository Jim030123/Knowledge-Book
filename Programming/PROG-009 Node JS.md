# Releases
![[Pasted image 20260120225501.png]]
- Major Node.js versions enter Current release status for 6 months, which gives library authors time to add support for them.
---

**After 6 months**
- Odd-numbered become unsupported
- Even-numbered move to Active LTS^[Long-term support that gurantees tha critical bugs will be fixed for a total of 30 months.] status.
---
- Production applications should only use Active LTS or Maintenance LTS releases.
