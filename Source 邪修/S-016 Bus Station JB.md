### 1️⃣ 主标题 & 站名（Taman Johor）

👉 很像：

- **Montserrat Bold**
- 或 **Poppins SemiBold**

特点：

- 圆角（rounded）
- 字母间距稍宽
- 很“现代 UI dashboard 风”

---

### 2️⃣ 时间 / 分钟（3 mins / 19 mins）

👉 很大概率是：

- **Roboto**
- 或 **SF Pro Display（类似 iOS）**

特点：

- 数字很干净
- “1”是直线，没有脚
- 适合做数据显示（dashboard）

---

### 3️⃣ 小字（JB Sentral / Larkin Sentral）

👉 通常是：

- **Roboto Regular**
- 或 **Open Sans**

特点：

- 易读
- 比较窄一点
- UI 常用

---

### 4️⃣ 特殊标签（ARRIVING）

👉 多半是：

- **Montserrat / Poppins ALL CAPS**

特点：

- 全大写
- 稍微加粗
- letter spacing 比较大（tracking）
