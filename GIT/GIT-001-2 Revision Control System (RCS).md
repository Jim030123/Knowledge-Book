# What is RCS
- Automates the storing, retrieval, logging, identification and merging of revisions.
- Useful for text that is revised frequently, including source code, programs, documentation, graphics, papers and form letters.
- Mostly superseded by Git and other heavyweight [[GIT-001-1 Version Control (VC)#Centralized VCS||VCS]], it still has uses a lightweight occasional system for controlling a small number of files.
